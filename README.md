Todo Lite App:
A simple full-stack task tracker with two states: Pending and Done.
Users can add tasks, view them, and toggle their status.

Features:
-Add new tasks
-View all tasks
-Toggle task status (Pending - Done)
-Works on Web and Mobile


Tech Stack:
-Backend: Node.js, Express.js
-Database: MongoDB Atlas
-Web App: React 
-Mobile App: Flutter

Deployment:
Backend - Render
Frontend - Vercel
